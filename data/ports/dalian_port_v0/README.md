# 大连港水域巡检原型 v0

本目录提供一个启发式的港口水域 / 水面巡检研究原型，用于验证 UAV-USV 异构协同巡检的任务建模、规则调度 baseline 和可视化流程。

重要边界：

- 本场景是大连港水域启发式原型地图，不是精确航海图，不能用于真实导航或海事作业。
- 本研究只涉及港口水域 / 水面巡检，不涉及堆场、道路、闸口、仓库、集卡运行区等港口地面巡检。
- 陆地、码头、岸壁、堤岸在本数据集中只作为水域边界或不可通行障碍，不作为巡检对象。

## 文件

- `dalian_port_v0_grid.json`：50x50 栅格水域原型图，包含水域、障碍、风险等级、depot 和元数据。
- `dalian_port_v0_tasks.json`：点、线、面三类水域巡检任务样例。
- `preview.png`：由 `tools/render_dalian_water_scenario.py` 生成的场景预览图。

## 任务含义

- 点任务：浮标 / 灯标、泊位前沿关键点、疑似漂浮物、疑似污染点、异常船舶停留点等，需要访问目标点或邻近水域。
- 线任务：主航道、进出港航路、泊位前沿线、岸线邻近水域等，需要沿离散线段序列巡视。
- 面任务：港池、锚地 / 待泊区、泊位前沿高风险水域等，需要进行局部覆盖巡检。

当前版本使用规则型 UAV-USV 调度 baseline。已有 MCPP 模型后续可作为面状水域任务的局部覆盖执行器接入，用于 `area_tasks` 中港池、锚地和高风险水域的局部覆盖。
