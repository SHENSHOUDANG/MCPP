# 上海港洋山水域巡检原型 v1

本目录是基于公开地图结构抽象的上海港洋山深水港水域巡检研究原型。它不是精确航海图，不用于真实导航、避碰或海事作业，只用于 UAV-USV 异构协同水面巡检任务建模、调度 baseline 和后续 MCPP 面任务接入验证。

## 真实地图依据与简化方式

公开资料显示，洋山深水港是上海港的重要深水海港部分，位于杭州湾外海小洋山一带，通过东海大桥与上海浦东相连；港区包含长条形集装箱深水泊位、填海/岛礁形成的港区陆域、能源/LNG 邻近水域、进出港深水航道和外侧锚地/待泊水域。

公开依据包括：Yangshan Port / 洋山深水港资料、东海大桥资料、洋山深水港集装箱码头资料，以及 DJI Matrice 350 RTK 公开规格。当前栅格没有直接下载或复刻海图/GIS 边界，而是把这些公开空间要素抽象为可运行的研究栅格。

本原型据此抽象出以下水域结构：

- 长条形集装箱码头陆域：作为不可通行障碍，只定义其前沿水域为巡检对象。
- 自动化码头/能源码头附近水域：设为高风险巡检区域。
- 东海大桥进港方向水域：设为桥区邻近巡检线和中风险水域。
- 深水进出港主航道：设为高风险线任务和航标点任务。
- 外侧锚地/待泊区：设为中高风险面任务。

## 点、线、面任务定义

- 点任务（point task）：目标是单个水面位置或邻近可航格，例如航标/灯标、疑似漂浮物、疑似污染点、异常船舶停留点、桥墩邻近水域检查点。执行形式是访问目标格并完成短时观测或确认。
- 线任务（line task）：目标是有顺序的离散水域线段，例如深水进港航道、泊位前沿、东海大桥邻近水域、LNG/能源码头警戒边界。执行形式是按给定节点顺序巡视整条线。
- 面任务（area task）：目标是一个有边界的水面区域，例如港池、泊位前沿高风险水域、自动化码头外侧水域、外侧锚地。执行形式是局部覆盖巡检；当前 baseline 用规则型 boustrophedon，后续可由 MCPP 覆盖执行器替换。

## UAV-USV 异构性

本原型采用中国港口巡检中常见的两类平台抽象：

- UAV：工业级多旋翼无人机，参考 DJI Matrice 350 RTK 级别。特点是速度快、视野大、适合航道/泊位前沿快速筛查和点线任务；限制是续航相对短、近距离水面确认能力弱。
- USV：国内常见 5-7 m 级小型电动/混合动力巡检无人船。特点是贴近水面、续航长、适合污染/漂浮物确认和面状水域持续巡航；限制是速度慢，且只能在可航水域行动。

配置文件中的速度、续航、载荷和任务偏好是研究参数，不代表某一具体厂家型号的强绑定性能。当前 MCPP 模型后续应作为 `area_tasks` 的局部覆盖执行器接入。
